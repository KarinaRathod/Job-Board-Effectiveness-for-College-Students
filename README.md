# 🚀 AI Job Platform Intelligence Dashboard

An AI-powered Streamlit dashboard to analyze job search platform effectiveness and predict placement outcomes using machine learning.

---

## 📌 Features

- 📊 Interactive Dashboard (KPIs, charts, filters)
- 🎯 Platform Performance Analysis
- 📈 Outcome Distribution Visualization
- 🧠 Candidate Segmentation (KMeans Clustering)
- 🤖 AI Placement Prediction (Random Forest)
- 📊 Prediction Confidence Score
- 🧠 Explainable AI (Feature Importance)
- 💬 Built-in AI Assistant (basic chatbot)
- 📥 Export filtered dataset

---

## 🧠 Tech Stack

- **Frontend:** Streamlit
- **Data Processing:** Pandas, NumPy
- **Visualization:** Plotly
- **Machine Learning:** Scikit-learn
